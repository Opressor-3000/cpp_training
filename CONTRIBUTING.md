# Как внести изменения

1. Сделайте форк репозитория.
2. Создайте ветку: `git checkout -b feature/<new_branch_name>`.
3. Внесите изменения и создайте коммит.
4. Отправьте ветку: `git push origin feature/<new_branch_name>`.
5. Откройте Pull Request в GitHub.

## Требования
- Все тесты должны проходить.
- Соблюдайте стиль кода
- PR должен содержать описание изменений.
